dic = 
{
	key_A : value_A,   #mit komma zu trennen
	key_B : value_B
}

- Key kann auch tuple sein 
	- `(subkey_A_1,subkeyA_2)` 
- Methoden dazu 
	- prüfen, ob key in dic 
		- `key_C in dic` 
			- falls in: true 
			- falls nicht in: false 
	- löschen key 
		- `del dic[key_A]` 
	- Länge 
		- `len(dic)`
